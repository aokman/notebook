Houdini 的参数表达式用于在参数栏中通过数学公式、变量、属性和函数动态计算参数值。

## 1. 参数栏和表达式语言

参数表达式直接写在节点参数栏中。参数求值时，Houdini 会根据表达式当前的上下文动态计算结果，而不是只保存一个固定数值。

Houdini 参数表达式主要有两类语言：

- **HScript 表达式**：默认参数表达式语言，适合短公式、通道引用、变量和表达式函数。
- **Python 表达式**：功能更强，但写法更长，适合需要调用 Houdini Object Model 或复杂逻辑的参数。

普通参数联动和简单动画优先使用 HScript。复杂脚本逻辑可以切换为 Python。

## 2. 数学公式

最基础的参数表达式就是数学公式。它适合用来描述一个参数值和时间、帧号或其他参数之间的关系。

例如，可以把某个位置参数设置为：

```text
$T * 2
```

表示让该参数随着当前时间 `$T` 线性变化。

也可以把通道引用放进数学公式里：

```text
ch("scale") * 0.5 + 1
```

表示读取当前节点的 `scale` 参数，把它缩小一半后再加 1。

数学公式通常不会单独存在很久。实际使用时，它经常和变量、属性、函数组合在一起，让参数根据场景状态自动变化。

## 3. 变量

变量用于把时间、帧号、工程路径、当前节点等上下文信息带入参数计算。

### 3.1 播放

| `$` 变量 | 说明 |
| --- | --- |
| `$FPS` | 播放速度，单位为每秒帧数，由 Playbar 控制。 |
| `$FSTART` | 动画的起始帧。`$NFRAMES = $FEND - $FSTART + 1`。 |
| `$FEND` | 动画的结束帧。 |
| `$F` | 当前帧。常用于渲染图片文件名编号。 |
| `$FF` | 当前浮点帧号，保留到小数点后 6 位。 |
| `$oldFF` | Houdini 20 之前 `$FF` 的旧行为，保留 6 位有效数字。 |
| `$NFRAMES` | 动画总帧数。`$NFRAMES = $FEND - $FSTART + 1`。 |
| `$RFSTART` | Playbar 当前显示范围的起始帧。 |
| `$RFEND` | Playbar 当前显示范围的结束帧。 |
| `$T` | 当前时间，单位为秒。等于 `($F - 1) / $FPS`。 |
| `$TLENGTH` | 动画总时长，单位为秒。 |
| `$TSTART` | 动画起始时间，单位为秒。 |
| `$TEND` | 动画结束时间，单位为秒。 |

## 4. 属性

属性用于把几何数据带入参数计算。这里列出的是常见的几何局部变量写法，是否可用取决于具体节点和参数上下文。

| `$` 变量 | 说明 |
| --- | --- |
| `$PT` | 当前点编号。 |
| `$PR` | 当前面编号。通常只在按 primitive 求值的参数上下文中有意义。 |
| `$TX` | 位置属性的 X 分量。 |
| `$TY` | 位置属性的 Y 分量。 |
| `$TZ` | 位置属性的 Z 分量。 |
| `$PSCALE` | 点的统一缩放属性。该变量依赖属性的局部变量映射，不一定在所有节点中可用。 |

## 5. 表达式函数

表达式函数用于在参数中完成更具体的计算、查询和转换。函数可以接收数字、变量、属性、字符串、节点路径，也可以嵌套在其他函数或数学公式中。

基本写法是：

```text
函数名(参数1, 参数2, ...)
```

例如：

```text
ch("../grid1/tx") * 2
```

表示读取 `../grid1` 节点的 `tx` 参数，并把结果乘以 2。

表达式函数可以嵌套使用：

```text
fit01(rand($F), 10, 20)
```

表示先用当前帧 `$F` 生成 0 到 1 的随机数，再把结果映射到 10 到 20 的范围。

常见表达式函数类型：

- **数学计算**：`abs()`、`sqrt()`、`pow()`、`sin()`、`cos()`、`tan()`。用于绝对值、平方根、幂、三角函数等数学计算。
- **数值范围**：`fit()`、`fit01()`、`fit10()`、`fit11()`、`clamp()`。用于把数值从一个范围映射到另一个范围，或限制在最小值和最大值之间。
- **随机数**：`rand()`、`noise()`。用于生成随机值或噪声值。
- **通道和参数**：`ch()`、`chs()`、`chf()`、`cht()`、`chramp()`。用于读取其他参数、字符串参数、指定帧或指定时间的参数值。
- **字符串处理**：`strlen()`、`strcat()`、`substr()`、`strreplace()`、`tolower()`、`toupper()`。用于处理字符串长度、拼接、截取、替换和大小写转换。
- **数字和字符串转换**：`atof()`、`ftoa()`、`padzero()`、`fpadzero()`。用于数字和字符串之间的转换，或给数字补零。
- **条件判断**：`if()`、`ifs()`。根据条件返回数值或字符串。
- **几何查询**：`point()`、`prim()`、`detail()`、`vertex()`、`npoints()`、`nprims()`。用于从几何体中读取属性或查询点、面等数量。
- **包围盒和中心**：`bbox()`、`centroid()`。用于查询几何体的包围盒和中心信息。
- **节点和路径**：`opname()`、`opfullpath()`、`oprelativepath()`、`opinputpath()`、`opexist()`。用于查询节点名称、完整路径、相对路径、输入路径或节点是否存在。
- **外部命令和表达式求值**：`system()`、`execute()`、`eval()`、`evals()`。用于执行命令或把字符串作为表达式求值。
- **Python 表达式**：`pythonexprf()`、`pythonexprs()`。在 HScript 表达式中计算 Python 表达式，并返回数值或字符串。

常用示例：

```text
ch("tx")
```

读取当前节点的 `tx` 参数。

```text
chs("file")
```

读取字符串参数。

```text
padzero(4, $F)
```

把当前帧补成 4 位数字，例如 `0001`。

```text
clamp(ch("scale"), 0, 10)
```

把 `scale` 参数限制在 0 到 10 之间。

```text
bbox("../box1", D_XSIZE)
```

读取 `../box1` 的包围盒 X 方向尺寸。

## 6. 字符串参数中的表达式

数值参数会直接求值，但字符串参数默认会被当作普通文本。字符串参数中可以展开变量，例如：

```text
$HIP/render/test.$F4.exr
```

如果要在字符串里嵌入数学表达式或函数，需要使用反引号：

```text
frame`padzero(5, $F)`.pic
```

结果类似：

```text
frame00001.pic
frame00002.pic
```

## 7. 多行表达式

参数表达式通常用于编写简短的一行公式，用来表达一个参数值和其他值之间的关系。例如，可以把球体的 X 位置设置为：

```text
$T * 2
```

如果需要在参数表达式中编写比较复杂的脚本逻辑，可以考虑把该参数切换为 Python 表达式。不过，传统 HScript 表达式语言本身也支持多行表达式、代码块、`if` 语句、`for` 循环和 `while` 循环。

在 HScript 和 Python 表达式中，都可以在参数栏按 `Ctrl + E`，或者右键参数并选择 `Expression > Edit Expression`，打开多行浮动编辑器。

### 7.1 代码块

代码块是一组放在花括号 `{}` 中的子语句。

子语句之间必须使用分号 `;` 分隔。

子语句可以包括以下几类：

- **赋值语句**：等号右侧可以使用表达式语法和表达式函数。变量类型可以写 `float` 或 `string`。对于浮点变量，类型声明可以省略；对于字符串变量，必须声明为 `string`。
- **修改赋值**：可以使用 `j += 5;`、`i -= 1;` 这类原地修改写法。
- **自增和自减**：可以使用 C 风格的 `i++;`、`j--;`。
- **条件语句**：可以使用 `if...else if...else`。
- **循环语句**：可以使用 `for` 和 `while`。
- **返回语句**：使用 `return` 返回整个代码块的最终值。

赋值示例：

```text
{
    size = 0;
    float frame = ch("outputframe");
    string name = ch("name");

    return size;
}
```

赋值出来的变量只在最外层 `{}` 代码块内部有效。

`return` 右侧可以继续使用表达式语法和函数。当执行到 `return` 语句时，它的表达式结果会作为整个代码块的值。如果没有任何 `return` 被执行，整个代码块会返回 `0`。

一个代码块中可以有多个 `return`，也可以在条件分支中提前返回：

```text
{
    if (evalmode == 4) {
        float slope = (ch("../inputrangey") - miniframe)
                    / (ch("../outputrangey") - minoframe);
        return miniframe + ($FF - minoframe) * slope;
    }

    return 0;
}
```

`#` 字符可以开始一行注释。`#` 后面的本行内容会被忽略。

```text
{
    # This is a comment.
    float size = ch("scale");
    return size;
}
```

### 7.2 if 语句

HScript 表达式语言中有一个 `if()` 函数，形式如下：

```text
if(expression, true_value, false_value)
```

需要注意的是，`if()` 函数中的条件表达式、真值表达式和假值表达式都会被求值。

多行表达式中的 `if` 语句则不同。它会先判断条件，然后只执行其中一个分支。

`if` 语句的结构类似 C 语言：

- `if` 后面跟一个放在圆括号 `()` 中的布尔表达式。
- 如果表达式的结果非零，则被视为 `true`，执行对应语句或代码块。
- 可以有零个或多个 `else if (expr)` 分支。
- 可以有一个可选的 `else` 分支。

示例：

```text
{
    float size = 0;

    if ($F > 10) {
        size = 23;
    } else if ($F > 20) {
        size = 47;
    } else {
        size = 50;
    }

    return size;
}
```

### 7.3 for 循环

多行表达式中的 `for` 循环也类似 C 语言。

`for` 后面跟圆括号 `()`，圆括号中包含三个表达式，并用分号 `;` 分隔：

```text
for (i = 0; i < 10; i++)
```

三个部分的含义分别是：

- 第一部分在循环开始前执行，通常用于设置循环变量的初始值。
- 第二部分在每次循环前求值。如果表达式为真，继续执行循环体。
- 第三部分在每次循环体执行后运行，通常用于递增或递减循环变量。

循环体可以是一条语句，也可以是 `{}` 包裹的代码块。

在循环中可以使用 `break` 和 `continue`。

示例：

```text
{
    float out = 0;

    for (i = 0; i < 10; i++) {
        out += i;
    }

    return out;
}
```

### 7.4 while 循环

`while` 循环同样类似 C 语言。

`while` 后面跟一个放在圆括号 `()` 中的表达式，以及一条语句或 `{}` 代码块。每次循环开始时，如果表达式求值为真，也就是非零，就执行后面的语句或代码块。

在 `while` 循环中也可以使用 `break` 和 `continue`。

示例：

```text
{
    float k = ch("parm");
    float i = 0;

    while (i < k) {
        i++;
        if (i % 2 == 0) continue;
    }

    return i;
}
```
